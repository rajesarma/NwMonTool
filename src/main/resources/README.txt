Ensure Java (JDK # 8) version installed in your system.
Ensure maven installed in your system.
Ensure Java Path set in Enviroment variables.
Ensure Maven Path set in Enviroment variables.

In Command Prompt, execute the following commands
mvn clean
mvn compile
mvn package

Navigate to the target folder, execute the following command
java -jar nwmontool-0.0.1-SNAPSHOT.war

This application will start by after executing the above command.

In the browser use localhost:8090 to view the webpage
